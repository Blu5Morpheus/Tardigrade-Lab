# Lab data fixtures

## `ligo_strain_sample.npz`

Generated by `scripts/generate_ligo_fixture.py`. Contains:

- `signals` — `(500, 256)` whitened binary-merger chirp templates
- `glitches` — `(500, 256)` synthetic glitch shapes (sine-Gaussian + blip)
- `labels` — `(1000,)` 0 for glitch, 1 for signal

To (re)generate locally:

```bash
pip install pycbc numpy scipy
python scripts/generate_ligo_fixture.py --output data/ligo_strain_sample.npz
```

The script is reproducible — same flags → byte-identical `.npz`.

**Do commit** the `.npz` to the repo. Render's Python build doesn't have
PyCBC available by default, and we do not want to install it server-side
(it's heavy and the fixture is static).
